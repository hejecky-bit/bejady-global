# BEJADY 官网重构方案

## 一、导航结构（改版后）

```
Home | Products ▾ | Factory | Certifications | Case Studies | Insights | Contact
                      ↓
              Single Cylinder
              Double Cylinder
              Triple Cylinder
              Four Cylinder
              Six Cylinder
```

- **Blog → Insights（工厂视角）**：更专业，SEO 友好
- **新增 Factory**：工厂实力展示页
- **新增 Certifications**：认证资质页
- **新增 Case Studies**：客户案例页

## 二、首页内容布局（从上到下）

| 区域 | 内容 | 目的 |
|------|------|------|
| ① Hero | 产品大图/视频背景 + 品牌标语 | 第一印象 |
| ② 产品展示 | 产品图片 + 播放按钮（点开看视频）→ 点图片进详情 | 直观看到机器运作 |
| ③ 数字实力 | 20年 / 15000台 / 60国 / 200+员工 | 数据信任 |
| ④ 工厂实力 | 车间实拍图 + 简介 | 看到真实工厂 |
| ⑤ 认证资质 | CE / UL / FDA / RoHS Logo 墙 | 官方背书 |
| ⑥ 客户案例 | Logo 墙 + 合作故事 | 社会证明 |
| ⑦ 最新洞察 | 最新 3 篇文章预览 | 专业内容 |
| ⑧ CTA | 联系/询盘 | 转化 |

## 四、博客改名

```
旧：Blog / 博客
新：Insights / 工厂视角

每个文章含：
├── 行业数据（引用来源）
├── 工厂经验（真实案例）
├── SEO 关键词
└── FAQ Schema（GEO 优化）
```

## 五、FAQ 策略（GEO 优化）

```
每个产品页底部：3-5 个产品专属 FAQ
全局 FAQ 页：10-15 个客户常问问题
全部用 FAQPage Schema 标记

这样 ChatGPT 搜索"Gelato machine"时，
你的 FAQ 内容会被 AI 直接引用。
```

## 六、后台管理系统（Admin）

### 方案一：Sanity CMS（推荐）
```
优点：
├── 🖱️ 可视化编辑，不用写代码
├── 📱 手机也能管理
├── 🔄 保存即发布
├── 🆓 免费版够用
└── 📸 图片自动优化

需要：注册 Sanity → 配置 schema → 上线
时间：半天搞定
```

### 方案二：简单 Admin 面板
```
优点：
├── 自己控制，不用第三方
├── 简单够用
└── 集成在网站内

需要：开发后台页面 + 对接 GitHub API
时间：1-2 天
```

## 七、实施顺序

```
第 1 步：导航 + 首页重新布局
第 2 步：产品展示方式优化
第 3 步：Factory / Certifications / Cases 新页面
第 4 步：博客改名 + 内容策略调整
第 5 步：FAQ 优化（GEO）
第 6 步：接入后台管理系统
```
